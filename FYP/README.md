# Final Year Project – IoT Kitchen Fire Prevention System

## Overview

This project was developed using ESP32 microcontroller and Arduino programming to detect fire/smoke conditions and send real-time alerts via a mobile IoT platform.

## Technologies Used

* ESP32
* Arduino C/C++
* Blynk IoT platform
* Sensors (Smoke / Temperature)

## Features

* Real-time sensor monitoring
* Mobile alert system
* IoT-based communication between device and app

## My Contribution

* Designed and implemented embedded system code using Arduino C++
* Integrated ESP32 with IoT platform (Blynk)
* Tested sensor response and system reliability
* Developed and debugged communication between hardware and mobile interface

## Outcome

The system successfully demonstrates a basic IoT-based fire detection and alert mechanism suitable for smart home applications.
